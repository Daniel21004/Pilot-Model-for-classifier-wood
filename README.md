# Pilot-Model-for-classifier-wood
This is repository contains the code for the pilot test of the original MobileNetV3 model original for the classification of four wood species 

# Install dependencies
For install dependencies execute the follow command into project

```
pip install -r requirements.txt
```

# Launch project
For launch the project execute the follow command


```
python app.py
```
รณ
```
python3 app.py
```

# Visualize project
For view the project, open a web browser and insert the following URL in the addressbar

```
http://localhost:7860/
```

If the prot 7860 is in use, modify it in the `interface.launch` method (at the end of code) of the 'app.py' file
